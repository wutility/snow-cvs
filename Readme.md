### CDN
```html
<script src="https://cdn.jsdelivr.net/npm/canvas-snow@1.0.0/index.min.js"></script>

```

## Usage
```
import Snow from 'canvas-snow';
```

## Methods & Examples

- **getCanvas(): HTMLCanvasElement**    
```html
Snow.getCanvas()
```
